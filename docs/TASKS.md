# Tasks — Chat-with-Bluey

## Sprint 1 — DB, Demo Data, Chat Engine
**Goal:** App loads with seeded conversations; user can send a message and get an AI reply stored in DB.

- [ ] Run migration SQL on Supabase (tables + seed data)
- [ ] Next.js project setup, Supabase client, env vars
- [ ] Chat list screen: loads `conversations` table, large text, high-contrast
- [ ] Conversation screen: loads `messages` for selected convo
- [ ] `POST /api/chat` route: inserts user message → calls OpenAI → stores assistant reply
- [ ] Daily quota check in `/api/chat` (reads/writes `daily_usage`)
- [ ] Empty state: "Start your first chat with Bluey" prompt
- [ ] Loading spinner on AI reply; error toast if OpenAI fails
- [ ] All screens render without login

**Definition of Done:** Open app → see 3 demo chats → open one → send "What can I cook with eggs?" → AI reply appears → row exists in `messages` in Supabase.

- [ ] `POST /api/reminders` CRUD routes

- [ ] Stripe Checkout session created via `POST /api/checkout`
- [ ] Stripe webhook `POST /api/stripe/webhook` → updates `subscriptions.status`

**Definition of Done:** Ask 5 questions → 6th shows paywall → complete Stripe test checkout → 7th question succeeds → `subscriptions.status = 'active'` in DB.
